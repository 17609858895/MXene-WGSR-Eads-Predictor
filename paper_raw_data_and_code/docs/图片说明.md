# 图片说明

说明：图2-图8已统一为 Arial 字体、粗体坐标轴标题与刻度、统一色系，并移除图内小标题及“a/b/c”式面板标签。论文正文或图注中可按下面说明引用各面板。

## 图2 数据集标签可用性与特征关联概览

左上：WGS 数据集中吸附能、反应能垒、反应能和反应路径标签的有效样本数。  
中上：吸附能 E$_{ads}$ 的整体分布，虚线表示中位数。  
右上：不同吸附物对应的 E$_{ads}$ 分布差异。  
左下：E$_{ads}$ 与关键数值描述符的 Pearson 相关矩阵。  
中下：E$_{ads}$ 与关键数值描述符的 Spearman 相关矩阵。  
右下：关键描述符之间的归一化互信息矩阵。

## 图3 特征筛选与冗余控制

左：基于置换重要性的关键特征排序，横轴为平均 R$^2$ 降低量。  
中：高重要性数值特征的 Spearman 冗余聚类，用于识别共线特征簇。  
右：Pearson、Spearman 与互信息三种筛选准则形成的共识特征集合。

## 图4 六种机器学习模型的吸附能预测基准

上方六个模型按行从左到右排列，分别展示 RF、ExtraTrees、GBR、HGBR、SVR 和 MLP。每个模型块左侧为 DFT E$_{ads}$ 与预测 E$_{ads}$ 的一致性散点图，空心圆表示训练集，实心三角表示测试集，虚线为 y = x，灰色带表示 ±0.50 eV 误差区间。每个模型块右侧为同一模型的预测误差分布窄图，用于快速比较残差偏移和离散程度。  
下方左侧：所有基础模型与 TOPSIS 集成模型在 hold-out 测试集上的 R$^2$、RMSE 和 MAE 对比。  
下方右侧：ExtraTrees 模型的学习曲线，展示训练样本数增加时训练集和验证集 R$^2$ 的变化趋势。

## 图5 TOPSIS 集成权重与统计显著性比较

左：各基础模型在 TOPSIS 集成中的客观权重。  
中：基础模型与集成模型在测试集上的绝对误差分布。  
右：模型两两比较的 Wilcoxon signed-rank 检验结果，颜色表示 $-\log_{10}(p)$。

## 图6 最优模型解释性分析

左：类 SHAP 局部贡献图，横轴为相对于参考输入的 E$_{ads}$ 预测变化，颜色表示特征取值由低到高。  
中上：全局置换重要性排序。  
右上：重点连续特征的部分依赖曲线，用于观察平均预测响应。  
中下：同一特征的累积局部效应曲线，用于降低特征相关性带来的解释偏差。  
右下：重点特征与交互特征着色的依赖散点图。

## 图7 机理敏感性与小样本反应能垒证据

左：先验机理关系图，表示组成、结构、电子态、吸附物和反应路径对 E$_{ads}$ 及反应能垒的潜在影响关系。  
中：关键连续特征增加 0.5 个标准差时对 E$_{ads}$ 预测的平均反事实影响及置信区间。  
右：具有反应能垒标签的 62 个样本中，E$_{ads}$ 与 E$_a$ 的相关性和低维趋势。

## 图8 掺杂体系外部迁移诊断与 DFT 标注趋势

左上：WGS 训练得到的 TOPSIS 集成模型直接迁移到 dopants 外部域时，DFT E$_{ads}$ 与预测 E$_{ads}$ 的一致性。该面板用于诊断域偏移；负 R$^2$ 表示直接迁移失败，不能作为可靠外部验证。  
中上：不同掺杂元素对应的预测残差分布，用于识别系统性偏差和需要补充训练的掺杂子域。  
右上：按掺杂元素汇总的 DFT-window score，该得分由 dopants 表已有 DFT E$_{ads}$ 标签计算，表示接近适中吸附窗口的程度。  
下方：基于 DFT-window score 的周期表热图，用于展示已有 DFT 标注数据中的掺杂趋势；该热图不应解释为当前 WGS 模型已完成可靠掺杂筛选。
