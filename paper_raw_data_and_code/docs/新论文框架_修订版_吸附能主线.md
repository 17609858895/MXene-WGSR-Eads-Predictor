# 新论文设计方案（修订版）

## 修订原因

对 `Comprehensive Dataset on MXene Properties for Catalyst Design (2024).xlsx` 核查后发现：

- `WGS` 表共 600 行，其中 `E_ads_ev` 有 599 条有效标签，适合作为主预测任务；
- `E_activation_ev` 和 `E_reaction_ev` 在 `WGS` 表中只有 62 条有效标签，只能作为小样本副任务或机理验证；
- `dopants` 表共 100 行，其中 `E_ads_ev` 有 94 条有效标签，但 `E_activation_ev`、`E_reaction_ev`、`Reaction` 没有有效标签，因此不适合做活化能迁移验证；
- 原框架中的“600+ 活化能预测”和“向 dopants 表迁移后仍保持活化能 R2=0.8x”缺乏数据支撑，需要调整。

因此，新论文主线应改为：**WGS-MXene 吸附能主预测 + 小样本活化能机制验证 + dopants 外部域迁移诊断与 DFT 标注掺杂趋势分析**。注意：当前结果显示 WGS 训练模型直接迁移到 dopants 域的表现较差，不能写成“外部泛化成功”或“模型已可靠筛选掺杂元素”。

## 拟定题目

**Domain-Aware Interpretable Ensemble Machine Learning for Adsorption Energy Prediction and Dopant-Domain Transfer Diagnostics of MXene Catalysts in the Water-Gas Shift Reaction**

中文：**面向水煤气变换反应 MXene 催化剂的吸附能预测与掺杂域迁移诊断：域感知可解释集成机器学习框架**

## 论文贡献

1. 基于 599 条 WGS-MXene 吸附能数据构建可解释机器学习模型，覆盖 H2O、CO2、H2、CO、OH、O2、O、H 等关键物种；
2. 引入元素物性编码、缺失指示变量和分组验证，降低材料重复与缺失填补导致的数据偏差；
3. 通过多模型比较、熵权-TOPSIS 集成和 Wilcoxon 检验，确定稳健的吸附能预测模型；
4. 用 SHAP/置换贡献、PDP、ALE 和因果敏感性分析解释关键结构-电子描述符；
5. 以 94 条 dopants 吸附能数据作为外部域诊断集，量化 WGS 模型在掺杂体系上的直接迁移失败，并利用 dopants 已有 DFT 标签进行吸附窗口趋势分析，为后续域自适应/补充 DFT 数据提供依据。

## 一、论文总体框架

### Abstract

强调 MXene 在水煤气变换反应中的催化潜力，吸附能是筛选吸附物-表面相互作用的核心指标。本文基于 600 条 WGS 数据和 100 条 dopants 数据，提出一个“特征工程-多模型比较-TOPSIS 集成-解释性分析-外部掺杂域诊断”的机器学习框架。模型可在 WGS 内部测试集中有效预测关键 WGSR 物种在 MXene 表面的吸附能，并揭示 Bader 电荷、形成能、MX 键长、表面终止基和吸附物类型等因素对吸附强度的主控作用。直接迁移到 dopants 数据时出现明显域偏移，因此 dopants 结果应作为迁移局限和 DFT 标注趋势分析，而不是高精度外部验证。活化能数据因样本有限，不作为主训练目标，而作为小样本机理一致性分析。

### 1. Introduction

- WGSR 的能源与工业意义；
- MXene 表面化学可调性及其催化应用潜力；
- DFT 高通量筛选的成本瓶颈；
- 现有研究以吸附能预测为主，但缺少域偏移诊断、掺杂外推风险评估和多视角解释；
- 本文贡献：域感知建模、TOPSIS 集成、可解释性、掺杂域迁移诊断、小样本能垒验证。

### 2. Dataset and Feature Engineering

- 数据来源：原论文及公开数据集；
- 数据结构：`WGS` 表作为训练/内部验证，`dopants` 表作为外部域迁移诊断和 DFT 标注掺杂趋势分析；
- 主目标：`E_ads_ev`；
- 副分析：`E_activation_ev`、`E_reaction_ev` 仅在 62 条有效样本上做描述性和敏感性分析；
- 特征类别：
  - 结构类：`a_ang`、`N_Layers`、`Layer_dist_MX`、`Layer_dist_MM`、`Length_MX`、`Length_MT1`；
  - 电子结构类：`E_Form`、`Bader_M1`、`Bader_M2`、`Bader_X`、`Bader_T1`、`Band_gap_PBE_ev`；
  - 组成类：`M1`、`M2`、`X`、`T1_type`、`Stacking`；
  - 吸附物类：`Mol`；
  - 掺杂域特征：`Dopant` 及其元素物性描述符；
- 缺失值处理：目标变量不进行填充；缺失值填充只用于输入描述符。数值描述符缺失用中位数填补，并加入缺失指示变量；类别缺失设为 `None`。
- 高缺失描述符控制：对 WGS 域缺失率 ≥50% 的输入描述符进行剔除敏感性分析，比较保留全部描述符与移除高缺失描述符后的模型性能，结果放入 Table S4。
- 正文 Methods 中只概括 descriptor 类别；详细 descriptor definitions、单位、缺失率和预处理策略放入 Table S1。

### 3. Methodology

- 三类相关性：Pearson、Spearman、互信息，形成线性/单调/非线性共识；
- 特征筛选：树模型重要性 + 层次聚类识别冗余特征；
- 模型库：RF、ExtraTrees、GradientBoosting、HistGradientBoosting、SVR、MLP；
- 验证方式：WGS 内部 train/test split + K-fold；dopants 表作为外部域压力测试。若 dopants R2 为负，应明确报告直接迁移失败；
- 缺失值敏感性验证：在相同 train/test split 下移除 WGS 域缺失率 ≥50% 的高缺失输入描述符，并重新训练同一组模型，检查 R2/RMSE/MAE 是否稳定；
- 集成策略：基于 R2、RMSE、MAE、MAPE 的熵权-TOPSIS 评分确定模型权重；
- 显著性检验：Wilcoxon 符号秩检验比较各模型绝对误差；
- 可解释性：置换贡献/SHAP 风格蜂群图、PDP、ALE、交互依赖图；
- 因果敏感性：基于先验 DAG 的随机森林敏感性分析，不声称严格因果发现；
- 掺杂域分析：比较不同 dopant 的预测残差，并基于 dopants 表已有 DFT 吸附能计算吸附窗口得分；不使用迁移失败模型的预测值作为可靠筛选依据。

### 4. Results and Discussion

- 4.1 数据分布与标签可用性；
- 4.2 相关性与共识特征；
- 4.3 特征筛选和层次聚类；
- 4.4 六模型性能比较，并在 SI 中报告高缺失描述符剔除敏感性；
- 4.5 TOPSIS 集成与统计显著性；
- 4.6 可解释性：主控描述符和非线性效应；
- 4.7 因果敏感性与小样本活化能一致性；
- 4.8 dopants 外部域迁移诊断与 DFT 标注掺杂趋势；
- 4.9 与原论文吸附能任务的差异化。

### 5. Conclusion

总结 WGS 吸附能主模型和关键机理解释；明确 dopants 直接迁移失败提示存在明显域偏移，不能把当前模型作为可靠掺杂筛选器；后续应补充掺杂域训练样本、加入显式 dopant 描述符或采用域自适应方法。活化能样本量不足，后续应补充 DFT 能垒数据用于真正的活化能迁移学习。

## 四、建议投向

- **Journal of Physical Chemistry C**：与原论文方向一致，适合强调数据集再挖掘和解释性；
- **Applied Surface Science**：适合吸附能预测和表面性质解释；

## 五、注意事项

1. 不应把 62 条 `E_activation_ev` 写成主模型的大样本结论；
2. 不应声称 dopants 表完成活化能迁移学习，因为没有活化能标签；
3. 目标变量不能进行缺失值填充；输入描述符填充必须与缺失指示变量和 Table S4 敏感性分析一起报告；
4. `Band_gap_HSE06_ev` 有效值极少，不宜作为主模型强依赖特征；
5. `M1_%`、`T1_%` 基本常数，对建模贡献有限；
6. 外部 dopants 验证应明确是吸附能域泛化，不是反应能垒验证。
